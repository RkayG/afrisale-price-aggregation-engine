oduct Name: Afrisale Pricing Engine
Type: Internal + Supplier-facing Web App
Goal: Replace spreadsheet-based price aggregation with a real-time, multi-supplier pricing system

1. 🎯 Problem Statement

Afrisale currently relies on spreadsheets to:

collect supplier prices
compare them
calculate final selling prices

This creates bottlenecks:

manual updates and errors
no real-time visibility
poor scalability with more suppliers/products
no proper access control
2. 🧭 Objectives
Primary Objective

Create a system where:

suppliers submit prices independently
the system automatically selects the lowest price
margin is applied
a live master price list is generated
Success Criteria
Suppliers can update prices without assistance
Admin sees all supplier prices in one view
Final price updates instantly after any change
Public list always reflects latest prices
3. 👥 Users
1. Admin (You)
full visibility
sets margin
manages products & suppliers
2. Suppliers (6–15)
submit/update prices
limited visibility (only their own data)
3. Public / Buyers
view final price list only
4. 🧩 Core Features
4.1 Supplier Portal

Access:

unique login link (no password)

Capabilities:

view assigned products
input/update prices
real-time save

Constraints:

cannot see other suppliers’ prices
cannot modify products
4.2 Admin Dashboard

Main View:
A table like:

Product	Supplier A	Supplier B	Supplier C	Lowest	Final

Capabilities:

view all supplier prices
highlight lowest price automatically
set global or per-product margin
override final price (optional)
add/edit products
manage suppliers
4.3 Pricing Engine (Core Logic)

For each product:

lowest_price = MIN(supplier_prices)

final_price = lowest_price + margin
OR
final_price = lowest_price * (1 + margin%)

Triggers:

supplier updates price
admin updates margin

Output:

instantly updated final price
4.4 Public Master Price List

Access:

shareable link

Displays:

product name
category
final price

Does NOT show:

supplier data
margin
internal calculations
5. 🔄 User Flows
Supplier Flow
Open unique link
View product list
Enter/update prices
System auto-saves
Exit
Admin Flow
Login
View dashboard
Adjust margin
Monitor price changes
Add/remove products or suppliers
Public Flow
Open link
View latest price list

price updates reflected in < 2 seconds
support up to:
15 suppliers
1,000+ products

low-cost infrastructure (initially)
